# mo-blog
- A fully-fledged blog system that supports create/read/update/delete posts.
- Team: Yuvraj Seegolam & Kaviraj Gosaye
- Assignment for web module

## Unauthenticated User
- View all posts
- Filter by category
- Read post
- Sign Up
- Sign In 

## Authenticated user
- View my posts
- Create new post
- Edit my posts
- Delete my posts
- Log out

## Stack
- Php
- XML as database
- XSD to validate XML
- XSLT for styling
- Ajax for asynchronous data fetching
